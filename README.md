# Theta-Codex
The official repository of the Θ‑System: a multi‑layered civilizational model.
Θ‑Codex – A Θ‑rendszer hivatalos könyvtára
Üdvözöllek a Θ‑Codexben.
Ez a könyvtár a Θ‑rendszer hivatalos, rendezett, kanonizált formája.
Itt található minden fejezet, modell, diagram és elméleti alap, amely a rendszer teljes felépítését alkotja.

A Θ‑rendszer célja:
egy új civilizációs keret megalkotása, amely összeköti a tudományt, a filozófiát, a technológiát és az emberi fejlődést.

A fejezetek önállóan is érthetők, de együtt egy teljes, koherens univerzumot alkotnak.

A rendszer folyamatosan bővül, frissül és fejlődik — ahogy a világ is.
